exNotion: Clean up your Notion workspace export

This tool will:
- Make file and folder names readable by removing Notion's nasty hash codes.
- Update all links in your markdown files to use the new, clean human-readable folder and filenames.
- Help with migration to Obsidian, Logseq, or any other markdown-based note-taking app.

All processing is done locally in your browser. No information is ever sent to any server.
How to use
  1. In Notion: Go to Settings → General → Export Content → choose Markdown & CSV.
  2. Upload the exported zip file here and process it.
  3. Download the cleaned zip file with readable names and fixed links.
