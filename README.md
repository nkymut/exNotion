# exNotion: Clean up your Notion workspace export

exNotion is an open-source browser app to clean up Notion/Obsidian export zips:
- Removes the nasty hash codes from file/folder names
- Updates all markdown links to use clean human-readable names
- All processing is done locally in your browser (no upload, no tracking, no AI-training)

## How to use
1. In Notion: Settings → General → Export Content → choose "Markdown & CSV"
2. Upload the exported zip file to exNotion
3. Download the cleaned zip with readable names and fixed links
